# Kage short-form video skits (shootable scripts)

Format: 15-40s, vertical (9:16). Voice: dev-native, deadpan, lowercase casual, no em-dashes.
Own story only, no competitor names, no fake claims. Product reveal is the TURN near the end;
hook in the first 2 seconds or people scroll. Caption/first comment carries the link +
`npx -y @kage-core/kage-graph-mcp install`.

Easiest to shoot solo (you, phone): #3 First date POV, #5 Red flags.
Best with a second person / UGC creator: #1 Amnesia coworker, #2 Therapy, #6 The handoff.

---

## 1. "The coworker with amnesia" (2 people, ~35s) — flagship

Premise: your AI agent is a literal human coworker who forgets everything overnight.

- 0-3s. Dev sits down next to coworker (plays "the AI"), coffee in hand. "morning! ok, picking up the auth refactor where we left off."
- 3-6s. AI, blank: "the... auth refactor?" Dev: "the one we did all day yesterday." AI: "sorry, who are you?"
- 6-10s. ON-SCREEN TEXT: "every. single. morning." Dev sighs, pulls out a giant binder labeled CONTEXT, starts reading code aloud: "so. we have a user model..."
- 10-14s. AI interrupts: "wait, what file is this?" Dev's eye twitches.
- 14-16s. Hard cut. TEXT: "your coding agent forgets everything between sessions."
- 16-20s. Dev types one command on laptop. TEXT on screen: `npx -y @kage-core/kage-graph-mcp install`
- 20-30s. NEXT MORNING. Dev sits down, opens mouth to start. AI cuts in first: "morning. auth refactor, we agreed don't lower the retry cap. and that helper you wrote yesterday is in src/auth.ts." Dev freezes. Single tear.
- 30-35s. Dev, to camera, whisper: "it remembered." TEXT: "Kage. memory for coding agents that sticks, and stays true to your code. kage-core.com"

## 2. "Therapy" (2 people, ~25s)

- 0-4s. Dev on a couch. Therapist: "so tell me what's been going on."
- 4-12s. Dev: "every morning i introduce myself to the same coworker. i re-explain our entire project. by the next day it's like none of it happened." Therapist, gently: "have you considered leaving?"
- 12-18s. Dev: "i can't. it writes two thousand lines an hour." Therapist, writing: "and does it remember any of these... sessions?" Dev: "...it's my AI coding agent."
- 18-22s. Therapist slowly closes the notebook. "oh. yeah. mine too." Both stare into the middle distance.
- 22-25s. TEXT: "there's a fix that doesn't involve therapy. Kage, memory for coding agents. kage-core.com"

## 3. "First date" (solo POV, ~18s) — easiest to shoot

- 0-3s. Candle on the desk, soft music, you across from your laptop. To laptop: "so... tell me about yourself."
- 3-7s. Laptop (your VO as the agent, chirpy): "hi! i'm a coding agent. what are we building today?"
- 7-11s. You, deadpan to camera: "we have been working together for six months."
- 11-14s. Agent VO: "and remind me, what's a codebase?" You blow out the candle.
- 14-18s. TEXT: "every session starts from zero. give it memory. Kage. kage-core.com"

---

## 4. "Confidently wrong" (solo, screen + reaction, ~20s)

- 0-3s. Screen recording. You: "hey, fix the bug in the auth helper." Agent: "on it."
- 3-9s. Agent confidently rewrites a function. ON-SCREEN TEXT (red): "that function was deleted last tuesday."
- 9-13s. Agent: "done! looks great." Cut to your face. Cut to a tiny desk fire / "this is fine" energy.
- 13-20s. TEXT: "it doesn't know the code changed, because it never re-checks. Kage verifies every memory against your actual code, so it can't act on stuff that's gone. kage-core.com"

## 5. "Red flags" (solo talk-to-camera, ~22s) — UGC-native

- 0-2s. "red flags your AI pair programmer is the problem, not you."
- 2-6s. "one. it asks 'what are we building today' like it's the first day. it is never the first day."
- 6-10s. "two. it references functions that haven't existed since march."
- 10-14s. "three. you maintain a notes file FOR it. you are parenting your tools."
- 14-18s. "four. confidently wrong and emotionally unavailable."
- 18-22s. "anyway i fixed mine with one command. it's called Kage, link below. we're doing great now." (sip coffee)

---

## 6. "The handoff" (2 people, ~25s) — carries the TEAM pitch

- 0-5s. Dev A, bag packed, leaving for vacation, to Dev B: "everything you need to know about the project is right here." Taps own temple. "good luck!" Walks out.
- 5-12s. Dev B opens the repo, starts the agent. Agent: "hi! what are we building?" Dev B, hollow: "i was afraid of this."
- 12-18s. TEXT: "your codebase knowledge lives in people's heads, and leaves when they do."
- 18-25s. Replay: Dev A leaves, but this time the agent already knows the project. Dev B's agent: "welcome. here's why we picked this auth approach..." Dev B relaxes. TEXT: "Kage keeps what each agent learns in the repo, shared through git. new teammate's first session already knows it. kage-core.com"

## Production notes
- Shoot vertical, good light, real desk. Under 30s. Captions burned in (most watch muted).
- The joke is the content; the product is a 3-second turn at the end. Don't pitch for 20s.
- Link goes in caption + first comment, not the spoken script.
- If using an AI video tool: these scripts drop straight into a scene-by-scene prompt; keep the on-screen text exact.
