# Background proxy lifecycle: parent-writes-after-confirm state, verified on every read

> kage up starts the proxy DETACHED by default mcp/vnext/runtime/proxy daemon.ts ; kage down stops it; kage run/status …

kage up starts the proxy DETACHED by default (mcp/vnext/runtime/proxy-daemon.ts); kage down stops it; kage run/status discover it. Design rules future agents must preserve: (1) STATE OWNERSHIP — the parent (kage up) writes .agent_memory/daemon/proxy.json only AFTER polling probeLocalPort confirms the detached child accepts connections; the child stays the untouched kage proxy command, so a failed start leaves no state, only the proxy.log to read. (2) VERIFIED READS — proxyDaemonState never trusts the file alone (the 860a272 lesson: SIGKILL leaves status files behind): it checks pid alive+ours+node (kill(pid,0) where ESRCH/EPERM mean not-ours, plus ps comm= includes node) AND the port accepts TCP; a record failing either check is removed and reported as stale_state_removed, and stopProxyDaemon never signals a pid from a stale record. (3) LOCATION — state lives in .agent_memory/daemon/ (plain legacy-daemon scratch dir), deliberately NOT in the 0700-leased vnext/ runtime dir, because the proxy runs on Node 18 and must not depend on a directory another component owns and secures. (4) A 0600/ownership check rejects checked-in or tampered state files (git delivers 0644, so a cloned hostile proxy.json is auto-distrusted) — untrusted files are neither believed nor deleted. (5) kage up --foreground writes no daemon state, so kage down does not manage it; a machine reboot needs one new kage up (no launchd/system service, stated in help/README instead of hidden).
Evidence: 29 new tests: proxy-daemon.test.ts covers dead-pid/live-pid-dead-port/malformed/untrusted state, start-confirm-then-write, never-listens reaped with no state, SIGKILL escalation; commands.test.ts adds an end-to-end run driving the real kage proxy child (up background -> verified state -> run auto-discovery -> down kills pid + cleans). Full suite 784/784 + dogfood 12/12; kage check 0 confirmed drift. Commit 8983d2b.
