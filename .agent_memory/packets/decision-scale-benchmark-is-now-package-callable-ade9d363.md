# Scale benchmark is now package-callable

> Kage's synthetic memory scale benchmark moved from a standalone benchmarks/scale kage memory.mjs script into the pack…

Kage's synthetic memory scale benchmark moved from a standalone benchmarks/scale-kage-memory.mjs script into the package API as benchmarkMemoryScale, CLI command kage benchmark --scale, and MCP kage_benchmark mode memory_scale. The viewer benchmark report now includes a bounded 240-packet scale proof and the viewer surfaces Scale proof cards. Verified larger local run: 240, 1000, and 5000 packets all hit 100%; 5000 packets had 362ms median recall and 99.83% context reduction.
