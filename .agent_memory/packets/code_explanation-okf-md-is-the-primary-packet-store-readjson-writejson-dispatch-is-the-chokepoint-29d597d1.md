# OKF .md is the primary packet store; readJson/writeJson dispatch is the chokepoint

> Kage's canonical on disk memory format is now OKF concept docs .md , not JSON. Packets live as .md under .agent memor…

Kage's canonical on-disk memory format is now OKF concept docs (.md), not JSON. Packets live as .md under .agent_memory/packets/ (and pending/). The KEY implementation insight: rather than convert every packet read/write call site, the dispatch lives inside the generic helpers readJson/writeJson (mcp/kernel.ts ~2512): if a path ends in .md, readJson routes through okfConceptToPacket and writeJson through packetToOkfConcept; otherwise JSON. This makes every packet I/O site (capture/writePacket, supersedeMemory, gc/compact, repair, personal sync, feedback, approvePending, diff-proposal) format-aware in ONE place. Legacy .json packets stay readable (dual-read via isPacketFile = endsWith .md||.json). Other load-bearing edits: packetFileName→.md; loadPacketsFromDir/loadPacketEntriesFromDir/graphFastFingerprint filters use isPacketFile; the merge driver mergePacketFiles + resolveConflictedPacket parse each conflict side as OKF-or-JSON; isReviewableMemoryPath regex includes .md (diff-proposal/sync depend on it); the staleguard porcelain glob uses isPacketFile; .gitattributes + repo .gitignore un-ignore packets/*.md (CRITICAL: if .gitignore only un-ignores *.json, .md packets are ignored and git-sharing breaks). GOTCHA: okfConceptToPacket now returns null for content containing git conflict markers (/^(?:<{7} |={7}\s*$|>{7} )/m) so validate/repair flag a conflicted packet instead of silently picking the ours-side kage-state. TEST GOTCHA: tests must NOT JSON.parse/JSON.stringify packet files directly — use the OKF-aware helpers parsePacket()/writePacketFixture() (added to kernel.test.ts; mcp.test.ts and trajectory.test.ts import okfConceptToPacket). The round-trip stays lossless via the embedded ```json kage-state block. Verified: 360 unit + 12 dogfood tests pass. Commit d26bbce on master.
Evidence: npm test → 360 pass + 12 dogfood pass, 0 fail; migrated 294 repo packets to .md; kage okf lint/verify/gains/recall work on the flipped store; pushed e96fba2..d26bbce to origin/master.
