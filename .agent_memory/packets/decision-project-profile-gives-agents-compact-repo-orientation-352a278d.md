# Project profile gives agents compact repo orientation

> Kage now has an external memory tool inspired project profile implemented as a rebuildable Kage report, not a separat…

Kage now has an Kage-native project profile implemented as a rebuildable Kage report, not a separate mutable memory store. kageProjectProfile summarizes existing structural index, code graph, and approved repo memory into repo totals, languages, top code+memory concepts, key files, memory focus, run commands, next actions, and warnings. It is exposed through CLI kage profile, MCP kage_profile, REST GET /kage/profile, local viewer report generation, and the dashboard. The profile tolerates older packets that lack quality/freshness fields so existing repos keep working.
