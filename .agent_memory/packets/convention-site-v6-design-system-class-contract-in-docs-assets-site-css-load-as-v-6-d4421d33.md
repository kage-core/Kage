# Site v6 design system: class contract in docs/assets/site.css (load as ?v=6)

> docs/assets/site.css is the v6 "measured receipts" design system for all five site pages index, guide, benchmarks, re…

docs/assets/site.css is the v6 "measured receipts" design system for all five site pages (index, guide, benchmarks, releases, demo). Every page must load it as assets/site.css?v=6 and build against the class contract documented in its header comment: layout .wrap/.wrap-narrow/.section/.section-alt/.section-head; nav .skip-link/.nav/.brand/.nav-links/.nav-cta/.gh-btn (+.gh-sep/.gh-star, star count span id=gh-stars filled by assets/stars.js); type .eyebrow/.lede/.accent; grids .grid-2/.grid-3/.grid-4 (aliases .cards-2/.cards/.cards-4 kept); .card; buttons .actions/.button/.button.primary/.cta panel; the signature receipt component .receipt > .r-head/.r-big (b + .unit)/.r-lines/.r-line (label span, span.dots leader, b value with .gain or .warn)/.r-foot with a .measured tag chip in the head (.estimate is the amber variant — estimated figures must be labelled); terminals .term/.term-head (aliases .terminal/.terminal-head kept) with .ps1 prompt and .dim comments; .cmd chip; .compare-scroll/.compare table; .stat-strip/.stat-chip; .footnote for honest-limits fine print. Removed from v5: .pflow/.pnode/.parrow, .demo-frame/.demo-cap, .receipt .r-hero (renamed .r-big; old .big/.dollars/.unit-inline structure gone) — pages still using those must migrate. Accessibility conventions: every page starts with <a class="skip-link" href="#main">, one h1 per page, [id] elements get scroll-margin-top so anchors clear the sticky nav, :focus-visible is styled globally, prefers-reduced-motion is respected. Nav order on every page: Home / Guide / Benchmarks / Releases / Demo + GitHub pill + .nav-cta.
Evidence: Rendered docs/index.html from file:// in the browser pane at full height: all sections (hero receipt, loop cards, wires, gate receipts, digestion, team, quickstart, footnote, footer) render correctly against the new CSS. Tag-balance parse clean. Committed as 2331d55.
