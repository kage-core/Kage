# Pivot decision: Kage → Probe (deterministic blast-radius PR gate)

> Decision 2026 06 27 : pivot Kage away from "verified memory for coding agents" a category being commoditized free by …

Decision (2026-06-27): pivot Kage away from "verified memory for coding agents" (a category being commoditized free by Claude Code / Copilot / Windsurf native memory, with no monetization surface and ~0 conversion) to **Probe — the deterministic, cross-vendor blast-radius PR gate**. Probe traces a diff's blast radius through the existing code graph and posts ONE non-LLM CI status check: "this change silently breaks these N callers across M files," each cited file:line, reproducible (same hash every run). Why this won: of 6 pivot theses scored by independent VC + skeptical-operator evaluators, Probe (pr-truth-gate) was the only candidate both judges rated "survives" (avg 68.5), because it uniquely satisfies all four binding constraints — weeks-to-first-dollar self-serve wedge, a real gateable monetization surface (hosted blocking check + evidence store, ~90% margin, no LLM on the hot path), a moat in the deterministic graph (not an LLM step a 50-line Action can clone), and cross-vendor neutrality (a gate that judges any agent's output can't be the agent's own vendor). Wedge artifact: `npx probe <PR-url>` → shareable self-signed scorecard (blast radius + "false-positive audit" of the team's existing AI reviewer + doc-lies the diff introduces). Business model: open-core — local engine/CLI free (MIT, funnel + reproducibility proof), hosted Probe Gate paid at ~$30/PR-opening-dev/mo (Team) → $60 (Business analytics) → $30-150K/yr Enterprise (EU AI Act evidence export, the 9-18mo expansion). Keep ~70-75% of the 35K-LOC engine (kernel.ts graph + impactSurface/shortestDependencyPath blast-radius walker, confidence-scored sub-0.5 edge suppression, truthReport/doc-lie + truthScorecardSvg, createSignedManifest, benchmark harness, 15-agent MCP plumbing); rip out the user-facing memory product surface (kage_learn/recall/supersede, MEMORY.md framing) and demote the .agent_memory packet store to an invisible internal evidence cache. Two decisive framing calls: (1) lead with BLAST RADIUS (pure graph, deterministic), NOT claim-vs-diff (secretly needs an LLM to parse PR prose → reintroduces false positives); (2) dev self-serve first, design-partner/regulated motion parked for month 9+. Full plan in PIVOT.md. GTM avoids cold DMs (proven ~0); artifact-first: Show HN + "Agent PR Honesty" leaderboard + GitHub Marketplace + MCP-directory listings.
Evidence: Two multi-agent workflows: (1) codebase asset inventory + live mid-2026 market research (10 agents) producing a grounding brief; (2) 6 pivot theses each scored by 2 diverse evaluators + red-team (19 agents). Probe was the sole candidate both evaluators rated survives=true; scoreboard: Probe 68.5, Reachable 63.0, Groundcheck 62.2, TruthGate 62.0, Provenable 61.0, Audit 59.5.
