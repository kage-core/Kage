# kage proxy spike: Headroom-style drop-in that injects + captures memory (form-factor unlock)

> Added mcp/proxy.ts + 'kage proxy' CLI command: a local HTTP proxy between any Anthropic API client Claude Code, Codex…

Added mcp/proxy.ts + 'kage proxy' CLI command: a local HTTP proxy between any Anthropic-API client (Claude Code, Codex CLI, aider) and api.anthropic.com, activated by one line (export ANTHROPIC_BASE_URL=http://localhost:8788). Motivation: Headroom (39K stars) proved the PROXY FORM FACTOR is the distribution unlock — zero per-agent install vs Kage's heavy 15-agent MCP+hooks matrix. This first slice is a SAFE passthrough: outbound it injects relevant verified memory into the request  (injectMemory, pure+unit-tested); inbound it taps the streamed response to record the exchange via observe(). It streams the model output back byte-for-byte. KEY BUG FOUND + FIXED: must strip hop-by-hop headers (transfer-encoding, connection, keep-alive, content-length, content-encoding) from the upstream response before writeHead, else the client never sees end-of-stream and hangs. Token COMPRESSION (the Headroom savings number) is deliberately deferred to a phase-2 'compress by remembering' layer — this slice only proves plumbing + the zero-config memory loop, so it currently ADDS tokens, not saves them. HONEST LIMITS: only works where base URL is settable (CLIs yes; Cursor/Copilot/closed IDEs no, so it covers FEWER agents than MCP hooks); it sees all traffic (trust ask); not yet hardened for prompt caching, anthropic-beta headers, or OAuth-vs-api-key; and NOT yet tested against real Claude Code + a real key (the actual go/no-go is a human live-task run, not the passing byte-level smoke test).
Evidence: End-to-end smoke test against a fake streaming upstream via a Node HTTP client (Claude Code's client shape): status 200, SSE intact (message_start->text_delta->[DONE]), memory injected outbound (upstream saw the 'injected by Kage' block), exchange captured inbound (1 observation), receipt printed '1 req · 1 injected · 1 captured · 1234 in / 42 out'. Unit tests: 3 injectMemory cases (string system, array system, no-match passthrough). Full suite 386 unit + 12 dogfood pass.
Verified by: node --test dist/proxy.test.js (3 pass); /tmp/proxy-smoke2.mjs Node-client end-to-end; npm test 386+12 pass
